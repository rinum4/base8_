Обработка демонстрирует работу с веб-сервисом RemoteManagement, который предназначен для удаленного управления запуском теста.

Для запуска теста необходимо сделать следующее:

  • Указать адрес к определению опубликованного веб-сервиса (WSDL). Формат адреса следующий: http://<АдресВебСервера>/<ИмяПубликации>/ws/RemoteManagement?wsdl
  • Указать имя пользователя и пароль. Пользователю должна быть назначена роль "ТестЦентр"
  • Нажать кнопку "Подключиться". При этом произойдет подключение к веб-сервису, и будет получен список запущенных клиентов
  • Назначить одному из клиентов роль "Агент ТЦ"
    • Выбрать клиента 
    • Указать дату начала
    • Указать дату окончания
    • Нажать кнопку "Добавить агента"
  • Назначить одному из клиентов роль "Управления тестом"
    • Выбрать клиента
    • Выбрать сценарий
    • Указать дату начала
    • Указать дату окончания
    • Нажать кнопку "Назначить сценарий"
