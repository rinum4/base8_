Форма предназначена для удаленного управления запуском теста.

Для запуска теста необходимо сделать следующее:

  • Назначить одному из клиентов роль "Агент ТЦ"
    • Выбрать клиента
    • Указать дату окончания
    • Нажать кнопку "Добавить"
  • Назначить одному из клиентов роль "Управления тестом"
    • Выбрать клиента
    • Выбрать сценарий
    • Указать дату начала
    • Указать дату окончания
    • Нажать кнопку "Добавить"

Клиенты должны быть предварительно запущены вручную
